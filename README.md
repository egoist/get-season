# get-season [![NPM version](https://img.shields.io/npm/v/get-season.svg)](https://npmjs.com/package/get-season) [![NPM downloads](https://img.shields.io/npm/dm/get-season.svg)](https://npmjs.com/package/get-season) [![Build Status](https://img.shields.io/circleci/project/egoist/get-season/master.svg)](https://circleci.com/gh/egoist/get-season)

> Get the current season of Anime world.

## Install

```
$ npm install --save get-season
```

## Usage

```js
const season = require('get-season')

season()
//=> 20161 : 现在在播 2016 年一月新番

season(true)
//=> 1 : 现在在播一月新番
```
